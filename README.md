## Installation
1. Lade das neueste **Engine-Paket** unter "Releases" herunter und entpacke es.
2. Lade das neueste **Update-Paket** (Artifact) herunter.
3. Entpacke das Update in denselben Ordner (Dateien Ã¼berschreiben).

## Starten
Doppelklick auf `START.bat`.

## Tasten
- **F8**: Setup (Ecken definieren)
- **F9**: Text scannen und vorlesen
- **F10**: Pause
- **F11**: Stop
